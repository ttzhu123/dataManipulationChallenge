The Seven Mental Models of Data Manipulation

The seven most important ways we manipulate datasets are:

  1. Assign: Add new variables with calculations and transformations
  2. Subset: Filter data based on conditions or select specific columns
  3. Drop: Remove unwanted variables or observations
  4. Sort: Arrange data by values or indices
  5. Aggregate: Summarize data using functions like mean, sum, count
  6. Merge: Combine information from multiple datasets
  7. Split-Apply-Combine: Group data and apply functions within groups

Tipüí° Method Chaining Tip for New Python Users

Why use lambda df:? When chaining methods, we need to reference the current state of the dataframe. The lambda df: tells pandas ‚Äúuse the current dataframe in this calculation.‚Äù Without it, pandas would look for a variable called df that doesn‚Äôt exist.

Alternative approach: You could also write this as separate steps, but method chaining keeps related operations together and makes the code more readable.

TipYour Answer: Drop vs Filter Strategies

.drop() specify columns or rows to remove, everything else stays. .filter() specify columns or rows to keep, everything else is removed. If most columns are useful and I only want to remove a few, I use .drop(). If I only want to keep a small, specific set of columns, I use .filter().

TipYour Answer: Handling Missing Data

.dropna() without subset, drops any row that contains a missing value in any column. .dropna(subset=[‚ÄòplannedShipDate‚Äô, ‚ÄòactualShipDate‚Äô]), drops rows only if plannedShipDate or actualShipDate is missing, missing values in other columns do not matter. I want to be selective because not all missing values matter for my analysis. Some columns may be optional, so missing values there should not cause rows to be dropped. I should only check columns that are critical to the analysis, not every column in the DataFrame.

TipYour Answer: Index Management

We use .reset_index(drop=True) after sorting to clean up the index.It makes the DataFrame easier to read and safer to work with after sorting.
